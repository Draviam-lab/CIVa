Chromosomal Instability aiding Variants (CIVa) Database


The CIVa database curates Loss of Function (LoF) variation predictions in chromosome segregation genes from multiple human genome sequencing efforts, including ELGH, BiB, Birm, COSMIC and Gnomad, in over 150,000 healthy individuals and patients.

The first version was published as part of Asifa et al., 2021, DOI to be included.

  • ELGH: East London Genes and Health Database. (sample size = 3781)
  • BiB: Born in Bradford. (sample size = 2624)
  • Birm: Birmingham (sample size = 1060). Assess date = 29 Jan 2020.
  • COSMIC: Catalogue of Somatic Mutations in Cancers Database. Assess date = Dec 2021
  • gnomAD: Genome Aggregation Database. Assess date = Dec 2021
  • Consequence: Predicted consequence of the mutation based on the transcript indicated.
  • Protein position: Position of amino acid affected.
  • Amino acid: Amino acid change e.g., R/* means Arginine changed into stop codon.
  • AF: Allele frequency: Frequency at which the allele is present in the database indicated.
  • N_Hom: Number of homozygous variants identified in the database indicated.
  • N_Het: Number of hetrozygous variants identified in the database indicated.
  • NA: Not Found.
